Installation

  • First of all you have to install Firefox.
  • Now you can install Greasemonkey. Greasemonkey allows you to customize the way a webpage displays using JavaScript.
  • It's now time to restart Firefox!
  • Then came back here and install GcTour. It appears a Greasemonkey dialog where you have to click install.
  • After that you are now ready to visit www.geocaching.com and see GcTour in action!
